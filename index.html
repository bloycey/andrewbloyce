Notable Projects

Github Portfolio | Node.js + Express + React

https://github.com/bloycey/github-portfolio

This project uses the Github API to organise repositories in a more intuitive way. It turns a users github page into an interactive portfolio. In creating this app I utilised a Node.js server to keep my Github credentials secure and used express for routing and as a webserver. On the front end I used React along with React Router to display the data quickly and neatly. This project greatly improved my skills in building APIs, working with existing APIs and managing the flow of data between the server side and the front end.

Product Watch | Node.js + Express + React + Local Storage

https://github.com/bloycey/product-watch

This web-app allows users to watch products online and track pricing changes. These changes are displayed in a graph along with other important product information including descriptions and images. The Node.js server does most of the work here in scraping the websites periodically and looking for changes. The front end is handled using React and the product data is persisted using HTML5 LocalStorage.
